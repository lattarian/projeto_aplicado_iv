# Projeto Aplicado IV – Análise da Taxa de Desocupação no Brasil (PNAD Contínua)

Repositório com os notebooks e arquivos de apoio do **Projeto Aplicado IV – Ciência de Dados EaD (2025/02)**, cujo objetivo é analisar a trajetória da **taxa de desocupação no Brasil** (e UFs selecionadas) ao longo do período recente, utilizando dados oficiais da **PNAD Contínua / IBGE (Tabela 4099 – Taxa de desocupação, média trimestral)**.

O foco do projeto é aplicar técnicas de **Análise Exploratória de Dados (EDA)** e **séries temporais (STL, ACF/PACF, ARIMA)** para compreender padrões, sazonalidade e tendência do desemprego, relacionando os resultados ao **ODS 8 – Trabalho decente e crescimento econômico**.
